# Northwind Sales Dashboard (Power BI)

This project analyzes retail sales performance using the Northwind dataset.

## Dashboard Insights
- Total sales and profit analysis
- Sales trend over time
- Revenue by country
- Category contribution
- Product performance analysis

## Tools Used
- Power BI
- DAX
- Data Modeling
- MySQL
